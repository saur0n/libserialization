## libserialization

This library allows to serialize standard C++ data types to binary form.

The following types are supported:
* `bool`
* `char`
* `int8_t`
* `uint8_t`
* `int16_t`
* `uint16_t`
* `int32_t`
* `uint32_t`
* `int64_t`
* `uint64_t`
* `long long` (not the same as `int64_t`)
* `unsigned long long` (not the same as `uint64_t`)
* `const char *` (C-strings)
* fixed-size arrays
* `std::array`
* `std::basic_string`
* `std::vector`
* `std::pair`
